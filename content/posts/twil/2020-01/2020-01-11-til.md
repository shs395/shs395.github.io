### 오늘 한 일

- 이번 주말에는 정렬에 대해 공부해 보려고 한다.
- 선택 정렬
- 그 동안 내가 써온 코드가 잘못되었음을 알았다.  

~~~c
// 그 동안 써온 코드
for(int i = 0; i < n - 1; i++){
  for(int j = i + 1; j < n; j++){
    if(list[i] > list[j]){
      temp = list[j];
      list[j] = list[i];
      list[i] = temp;
    }
  }
}
~~~

~~~c
// 좀 더 맞는 코드
for(int i = 0; i < n - 1; i++){
  int min = i;
  for(int j = i + 1; j < n; j++){
    if(list[j] < list[min])
      min = j;
  }
  temp = list[min];
  list[min] = list[i];
  list[i] = temp;
}
~~~

앞에 코드는 배열의 원소간의 자리 바꿈이 훨씬 많이 일어나기 때문에 조금 더 비효율적이다.
