- path.join

- __dirname
  - node 파일이 실행되는 곳이라고 한다. (index.js 같은 것)
  - ex) __dirname ==  /nodeFileFolder (뒤에 / 안붙음)
  
- branch 에 한 커밋은 카운트가 되지 않는다고 한다..

  - Open a pull request  to have your changes merged into the default branch or the gh-pages branch.
